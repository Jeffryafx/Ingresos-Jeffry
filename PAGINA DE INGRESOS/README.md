# 💰 Ingresos Jeffry

Sistema web de gestión de finanzas personales diseñado para ingresos variables e irregulares.

## 🎯 Características

- **Dashboard intuitivo** — Visualiza ingresos, gastos y balance en tiempo real
- **Sistema de Sobres Semanales** — Divide automáticamente cada pago en 6 categorías
- **Gestión de Gastos** — Registra gastos por categoría con gráficos de distribución
- **Plan Mensual de Ocio** — Calcula presupuesto para salidas con pareja
- **Datos Persistentes** — Todo se guarda en localStorage (navegador)
- **Exportable** — Descarga tu información como JSON para respaldo

## 📊 Cómo funciona

### Los 6 Sobres
Cuando recibes dinero, el sistema divide automáticamente en:

1. **Gastos Fijos** — Arriendo, servicios, comida base
2. **Deudas** — Cuotas y obligaciones (pagar primero)
3. **Ahorro** — 20% del dinero recibido (intocable)
4. **Ocio/Pareja** — 10% para salidas (tope fijo)
5. **Herramientas Trabajo** — 5% para software, cursos, equipos
6. **Libre** — Lo que sobra tras cubrir todo lo anterior

## 🚀 Uso

1. Abre `index.html` en tu navegador
2. Registra tu **meta mensual** en Configuración
3. Agrega **gastos fijos** en Configuración
4. Registra **ingresos** cuando recibas dinero
5. Registra los **gastos** en la categoría correspondiente
6. En **Sobres**, registra el dinero que recibiste esa semana
7. El sistema divide automáticamente en 6 categorías 📦
